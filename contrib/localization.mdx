RSS Guard supports localization into many languages through [Crowdin](https://crowdin.com/project/rssguard), a collaborative translation platform. All translation work happens there — no code changes or pull requests are needed to add or update a translation.

## How to contribute

To start translating RSS Guard:

1. Visit the [RSS Guard project on Crowdin](https://crowdin.com/project/rssguard).
2. Check the current translation status for your language. You can see what percentage of strings are already translated and how up-to-date they are.
3. Join the translation team for your language (or request a new language if yours is not listed).
4. Translate strings directly in the Crowdin editor. Changes are reviewed and merged into RSS Guard releases periodically.

<Tip>
  Even partial contributions help. Start with the most visible strings — menus, buttons, and settings labels — so users notice the improvement right away.
</Tip>

## Maintenance requirements

<Warning>
  All translators commit to keeping their translations up-to-date. If a translation is not updated regularly and fewer than **50% of strings** are translated, that translation and its team will eventually be **removed from the project**. Make sure you can commit to ongoing maintenance before starting a new language.
</Warning>

Keeping translations current matters because new strings are added with each release. An outdated translation with many untranslated strings degrades the experience for speakers of that language.

## Checking translation status

You can check the current status of all languages — including completion percentages and recent activity — at [crowdin.com/project/rssguard](https://crowdin.com/project/rssguard).
