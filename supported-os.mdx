---
title: "Supported Operating Systems"
description: "RSS Guard runs on Windows 7+, GNU/Linux, macOS 10.14+, BSD variants, and OS/2, including niche targets like PinePhone and Inkbox e-readers."
---

RSS Guard is a cross-platform application built on Qt, which allows it to run across a wide range of desktop and embedded operating systems. The table below lists all platforms known to work.

| Platform | Details |
| :--- | :--- |
| **Windows 7+** | All Windows versions from 7 onward. Two package flavors are available: `win7` (Qt 5) and `win10` (Qt 6). See [Downloads](/downloads) for details. |
| **GNU/Linux** | Mainstream distributions including Arch, Ubuntu, Manjaro, Debian, Fedora, and others. |
| **GNU/Linux — PinePhone** | Runs on PinePhone smartphone Linux distributions. |
| **GNU/Linux — Inkbox** | Ported to [Inkbox](https://github.com/Szybet/rssguard-inkbox), the OS for e-ink book readers. |
| **BSD** | FreeBSD, OpenBSD, NetBSD, and other BSD variants. |
| **macOS 10.14+** | macOS Mojave and later. DMG packages are provided. |
| **OS/2** | ArcaOS and eComStation. |

<Note>
  RSS Guard has also been ported to **KOBO e-readers** via a community project at [github.com/Szybet/rssguard-inkbox](https://github.com/Szybet/rssguard-inkbox). This port runs on Inkbox-based KOBO devices. Report bugs for this port to the port author upstream.
</Note>

For installation instructions specific to your platform, see the [Downloads](/downloads) page.
